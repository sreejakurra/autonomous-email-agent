# Autonomous Email Agent

An AI-powered email management system that automatically processes incoming emails, categorizes them, identifies action-required messages, and generates intelligent reply suggestions.

## Features

- Email Categorization
- Priority Detection
- Action Needed Detection
- AI Reply Generation
- FastAPI Backend
- Interactive Dashboard
- Local LLM Support via LM Studio

## Tech Stack

- Python
- FastAPI
- HTML/CSS/JavaScript
- Gmail IMAP/SMTP
- LM Studio
- REST APIs

This project is currently in active development.
